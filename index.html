Latest Project: Photo Gallery API

The photo gallery for JustJennette.com has received more traffic than the main domain. There was a need to preserve CPU usage and optimize for mobile. The backend is programmed with Coppermine, a PHP based script for photo galleries. However, there was no JSON API available.

I created a plugin called JSONFeed. This displayed albums and photos in a useable format for jQuery. With the help of Amazon CloudFront's servers, I was also able to cache the JSON feed. This helped two ways: receive the API faster and use less server resources.

View JustJennette Gallery

Jennapp: Celebrity based app for iPhone and Android

Since the release of JustJennette.com, there was a need for fans to stay current with Nickelodeon actress Jennette McCurdy from anywhere and receive notifications.

The mobile app released for iPhone and Android were built with Titanium SDK, a foundation to make code management easy and fast. Fans helped decide the name Jennapp, a clever use of the combination of Jennette and the word app. Jennapp features regularly updated news, photos, videos, sticker editing, and more. Notifications are also sent instantly to the fan with updated information.

Jennapp is available to download now for free.
